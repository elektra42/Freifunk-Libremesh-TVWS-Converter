AVR ATMega8 source files.

The purpose of this tool is to control the RF2051 mixers on board the
external Libremesh/Freifunk TV-Whitespace converter via serial port.

Use 3.3 V level RS232 at 115200 Baud, 8N1 with your favorite serial terminal
to set the registers of the mixers in the device or configure them via your
router with a script.
